# zorber

[![circleci-badge][]][circleci-target]

> A GitHub App built with [Probot](https://github.com/probot/probot) that Zorber rolls the orbs

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
